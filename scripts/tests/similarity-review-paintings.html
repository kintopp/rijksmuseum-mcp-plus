How to read this

Iconclass mode — finds artworks sharing the same Iconclass subject classifications. Score = sum of depth(notation) × IDF(notation) for all shared codes. Higher = more specific shared motifs.

Lineage mode — finds artworks with shared visual-style lineage (e.g. "after Rembrandt", "workshop of"). Score = sum of qualifier_strength × creator_IDF. Strength: after/copyist=3, workshop=2, circle/follower=1.

Depicted person mode — finds artworks depicting the same named persons (rulers, saints, artists). Score = sum of log(N / df) for each shared person. Rare persons score higher.

Intersections (yellow section) — artworks appearing in 2+ modes within the top 50 results each. Orange border = 2 modes, red border = 3 modes. These are the strongest multi-signal matches.

Click any image to open the artwork on the Rijksmuseum website. Each result card shows the shared motifs/lineage/persons. Ask: do these results look genuinely similar to the query artwork?
